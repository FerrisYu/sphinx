SimpleGIS API Reference¶

GeographicPoint¶

A GeographicPoint object describes a geographic location using latitude, longitude, and elevation.

GeographicPoint(latitude, longitude[, ...]) Construct a GeographicPoint object.
GeographicPoint.get_closestPoint(gp_list) Gets the closest GeographicPoint object in a list of GeographicPoint based on their 2D direct distance with the current GeographicPoint.
GeographicPoint.get_distance(other) Gets the 2D geographic distance between the other and this Geographic Point.
GeographicPoint.get_furthestPoint(gp_list) Gets the furtest GeographicPoint object in a list of GeographicPoint based on their 2D direct distance with the current GeographicPoint.
GeographicPoint.get_latitude() Gets the latitude of the current geographic coordinate.
GeographicPoint.get_locationInfo() Gets the location information for the current geographic coordinate.
GeographicPoint.get_longitude() Gets the longitude of the current geographic coordinate.
GeographicPoint.get_time() Gets the time when the point was taken.
GeographicPoint.now() Set this object with the 3D geographic attributes of the current positition, and the time when it’s taken.
GeographicPoint.toString() Return a string of that value it in degrees, minutes, seconds format.
class SimpleGIS.GeographicPoint(latitude, longitude, elevation=None, time=None)¶

Construct a GeographicPoint object.

Parameters:
  • latitude(float) – latitude of the geographic coordinate. Range: -90 ~ 90.
  • longitude(float) – longitude of the geographic coordinate. Range: -180 ~ 180.
  • elevation(float) – (optional) elevation of the geographic coordinate. None if this is a 2D geographic coordinates.
  • time(datetime) – (optional) The time when the point was recorded.
Raises:

ValueOutOfRangeError – Raise when latitude or longitude is out of range.
get_closestPoint(gp_list)¶

Gets the closest GeographicPoint object in a list of GeographicPoint based on their 2D direct distance with the current GeographicPoint.

Parameters:gp_list(GeographicPoint[]) – A list of GeographicPoint used to measure the 2D distance.
Return type:SimpleGIS.GeographicPoint 
>>> import SimpleGIS
>>> l = []
>>> l.append(SimpleGIS.GeographicPoint(10,20))
>>> l.append(SimpleGIS.GeographicPoint(0,0))
>>> gp = SimpleGIS.GeographicPoint(20,40)
>>> cp = gp.get_closestPoint(l)
>>> cp.get_latitude()
10
get_distance(other)¶

Gets the 2D geographic distance between the other and this Geographic Point.

Note

Distance is measure in two dimension. 3D geographic points are treated as 2D points.

Parameters:other(GeographicPoint) – The GeographicPoint used to measure the distance.
Return type:SimpleGIS.GeographicDistance 
>>> import SimpleGIS
>>> p1 = SimpleGIS.GeographicPoint(3,4)
>>> p2 = SimpleGIS.GeographicPoint(0,0)
>>> gd = p1.get_distance(p2)
>>> gd.get_direct_distance()
5

GeographicDistance¶

A GeographicDistance object describes the 2D distance between two geographic points.

GeographicDistance(d_latitude, d_longitude) Construct a GeographicDistance object
GeographicDistance.get_directDistance() Gets the direct_distance between the two geographic points in km.
GeographicDistance.get_latitude_diff() Gets the latitude difference between the two geographic points.
GeographicDistance.get_longitude_diff() Gets the longitude difference between the two geographic points.
class SimpleGIS.GeographicDistance(d_latitude, d_longitude)¶

Construct a GeographicDistance object

Parameters:
  • d_latitude(float) – latitude difference between the two geographic points. Range: 0 ~ 180.
  • d_longitude(float) – longitude difference between the two geographic points. Range: 0 ~ 360.
Raises:

ValueOutOfRangeError – Raise when latitude or longitude difference is out of range.

Note

You shouldn’t construct objects of this class yourself, instead, you will get the object from the get_distance method in GeographicPoint class.

Polygon¶

A polygon is an area surroundded by a list of geographic points, which are nodes of a polygon.

Polygon(gp_list) Construct a Polygon object.
Polygon.add(gp) Add a new point to the polygon.
Polygon.get_GPList() Return the stored list of geographic points
Polygon.get_GeoJsonFormat() Convert a polygon object to a valid GeoJson string.
Polygon.get_area() Return the area of the polygon.
Polygon.isValid() Check whether the current gp_list con construct a valid polygon.
Polygon.is_inbound(gp) Returns true if the geographic point is inside or on edge of the polygon, false if not.
Polygon.next() Return the next geographic point in gp_list.
Polygon.remove(i) Remove a point from the polygon
class SimpleGIS.Polygon(gp_list)¶

Construct a Polygon object.

Parameters:gp_list(GeographicPoint[]) – a list of geographic points which represense the nodes of the polygon.

Add a new point to the polygon.

get_GeoJsonFormat()¶

Convert a polygon object to a valid GeoJson string. GeoJon format is a valid format to export to google map.

Return type:str 
>>> import SimpleGIS
>>> l = []

get_area()¶

Return the area of the polygon.

>>> import SimpleGIS
>>> l = []
>>> l.append(SimpleGIS.GeographicPoint(3,0))
>>> l.append(SimpleGIS.GeographicPoint(0,4))
>>> l.append(SimpleGIS.GeographicPoint(0,0))
>>> p = Polygon(l)
>>> p.get_area()
6
>>> p.remove(SimpleGIS.GeographicPoint(0,0))
>>> p.get_area()
-1

Note

At least 3 geographic points are needed to get a polygon area. Polygon with least than 3 geographic points will return -1.

Track¶

A Track is an ordered list of geographic points. It represense a series of geographic points that are captured in order.

Track(gp_list) Construct a Track object.
Track.get_GPList() Return a list of geographic points
Track.get_GeoJsonFormat() Convert a track object to a valid GeoJson string.
Track.get_endTime() Return the recorded time of the last stored geographic point.
Track.get_point(i) Return the geographic point at index location.
Track.get_startTime() Return the recorded time of the first stored geographic point.
Track.get_totalDistance() Return the total distance traveled of the track in km .
Track.next() Return the next geographic point in the ordered geographic point list.
Track.push_back(gp) Add a new point to the end of the ordered list.
Track.remove(i) Remove a point from the polygon

GPXFParser¶

A parser class for parsing GPX file. It can parse GPX files into Track, Polygon, and GeoJon types.

GPXFParser(gpx_f) Construct a GPXFParser object.
GPXFParser.get_GPList() Parse a valid GPX into a list of geographic points.
GPXFParser.get_GeoJsonFormat() Parse a valid GPX into a valid GeoJson string.
GPXFParser.get_Track() Parse a valid GPX into a Track object.
GPXFParser.parse(gpx_f) Parse the input GPX file.
class SimpleGIS.GPXFParser(gpx_f)¶

Construct a GPXFParser object.

get_GPList()¶

Parse a valid GPX into a list of geographic points.

Return type:list 
>>> import SimpleGIS
>>> f = open("foo.gpx", "r")
>>> parser = GPXFParser()
>>> parser.parse()
>>> gp_list = parser.get_GPList()
>>> gp_list[0].get_time()
datetime(2015, 6, 10, 15, 8, 24, 78915)

LocationInfo.get_country() This gets the country of this location
LocationInfo.get_province() This gets the province of this location
LocationInfo.get_timeZone() This gets the time zone of this location
class SimpleGIS.LocationInfo(country, province, city, timeZone)¶

Construct a locationInfo object.

Parameters:
  • country(str) – country of this new location.
  • province(str) – province of this new location.
  • city(str) – city of this new location.
